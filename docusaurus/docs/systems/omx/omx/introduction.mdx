# About OMX

<img src="/img/systems/omx/quick_start_guide/main_image.webp" width="600"/>

The ROBOTIS **OMX** is an entry‑level Physical AI manipulator designed for hands‑on dataset collection and comparison, helping newcomers step into Physical AI with confidence. This compact 5‑DOF platform, built on **DYNAMIXEL‑X**, supports end‑to‑end IL/RL pipelines and **ROS 2‑native** workflows.

## Key Features

<div class="features">
  <div class="feature-item">
    <div class="feature-icon">⚙️</div>
    <h3>No Motor Setup Required</h3>
    <p>All motor parameters are preconfigured at the factory. You can connect immediately and start using the robot arm without any additional hardware or software setup.</p>
  </div>

  <div class="feature-item">
    <div class="feature-icon">🔌</div>
    <h3>No Calibration Required</h3>
    <p>Every DYNAMIXEL actuator is factory-calibrated, so users never need to perform calibration themselves. Both hardware and software are fully supported for plug-and-play use.</p>
  </div>

  <div class="feature-item">
    <div class="feature-icon">🛡️</div>
    <h3>Safe Homing Bring-Up</h3>
    <p>The follower arm can safely return to its initial pose from any position during bring-up. This ensures stable, predictable operation and minimizes risk for first-time users.</p>
  </div>

  <div class="feature-item">
    <div class="feature-icon">🧭</div>
    <h3>360° Base Rotation</h3>
    <p>The base motor uses an extended-position design for full 360° rotation. With extended cabling, it can rotate indefinitely and expand the usable workspace.</p>
  </div>

  <div class="feature-item">
    <div class="feature-icon">🪶</div>
    <h3>Light Weight</h3>
    <p>OMX is engineered to be extremely lightweight, making it easy to carry and deploy anywhere while maintaining robust performance during extended use.</p>
  </div>

  <div class="feature-item">
    <div class="feature-icon">✨</div>
    <h3>Low-Friction Leader</h3>
    <p>A special low-friction design makes the leader smooth and easy to operate. Fine movements feel natural with minimal effort, supporting intuitive data collection.</p>
  </div>

  <div class="feature-item">
    <div class="feature-icon">🔋</div>
    <h3>No Power Supply Required for Leader</h3>
    <p>Unlike conventional robot arms, the OMX-L leader works with a single USB-C connection, enabling simple laptop use for simulation or quick trials.</p>
  </div>

  <div class="feature-item">
    <div class="feature-icon">🎯</div>

    <p>A spring-assisted control mechanism provides comfortable, repeatable grip feedback, making long data-collection sessions easier and less fatiguing.</p>

## OMX Lineup Overview

<img src="/img/systems/omx/overview/omx_models.png" width="800"/>

:::info
The leader and follower are only available for purchase as a set.
:::

The OMX series offers multiple configurations for various teleoperation and manipulation tasks:

- `OMX-L`
  Leader unit for teleoperation. Provides an intuitive interface for remotely controlling other OMX units.

- `OMX-F`
  Gripper + 5‑DOF robot arm. Ideal for object manipulation and perception tasks.

- `OMX-AI`
  Complete teleoperation set including OMX-L leader unit and OMX-F follower unit.
  Provides a full leader-follower system for imitation learning and teleoperation applications.
