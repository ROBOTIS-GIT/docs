Select the active communications protocol for your DYNAMIXEL actuator.
In order to use Modbus-RTU you must update to firmware version V11 or higher.



| Value | Description                                                                                                |
|:-----:|:-----------------------------------------------------------------------------------------------------------|
|   2   | [DYNAMIXEL Protocol 2.0]                                                                                   |
|  10   | [Modbus-RTU, Industrial Standard Protocol]<br />![](/img/dxl/p/protocol_mode_modbus.png) |

[DYNAMIXEL Protocol 2.0]: https://emanual.robotis.com/docs/en/dxl/protocol2/
[Modbus-RTU, Industrial Standard Protocol]: http://modbus.org/docs/PI_MBUS_300.pdf


:::warning

**WARNING**: [DYNAMIXEL Wizard 2.0](https://emanual.robotis.com/docs/en/software/dynamixel/dynamixel_wizard2/) is required to change the communications protocol of your DYNAMIXEL servo, as R+ Manager 2.0 does not support Modbus-RTU communication.

:::
