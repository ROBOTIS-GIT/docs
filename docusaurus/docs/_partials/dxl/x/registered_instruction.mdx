Indicates whether an instruction has been registered by the [Reg Write Instruction](/docs/dxl/protocol/protocol2#reg-write-0x04) and is ready for execution. Following the execution of a registered action the Registered Instruction (69) value will automatically be updated to 0.

| Value | Description                                  |
|:-----:|:---------------------------------------------|
|   0   | No instruction registered by REG_WRITE.      |
|   1   | Instruction registered by REG_WRITE. |
