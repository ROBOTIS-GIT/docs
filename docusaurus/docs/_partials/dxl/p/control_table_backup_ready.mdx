The value in this address indicates whether the backup of the control table exists after sending the [Control Table Backup Packet](/docs/dxl/protocol/protocol2#control-table-backup-0x20).

| Value | Description                    |
|:-----:|:-------------------------------|
|   0   | The backup data doesn't exist. |
|   1   | A saved backup data exists.    |


:::note

**NOTE**  
Backup Ready is available from firmware **V12**.  
See [Backup and Restore](/docs/software/dynamixel_wizard_2_0#backup-and-restore) for more details.

:::
