## Data Classes

**ControlTableItem**

| Attribute | Value Type  |
| ----------| ----------- |
| address   | Int         |
| size      | Int         |

**StagedCommand**

| Attribute                 | Value Type                     |
| --------------------------| -------------------------------|
| command_type              | CommandType                    |
| id                        | Int                            |
| address                   | Int                            |
| length                    | Int                            |
| data                      | List[Int]                      |
| status_request            | List[StatusRequest] or None    |
| motor                     | Motor or None                  |
| allowable_operation_modes | List[OperationMode] or None    |
