# Install requirements

* [Download Java Development Kit (JDK) (`Linux 32-bit` or `Linux 64-bit`)](http://www.oracle.com/technetwork/java/javase/downloads/index.html)

  ![](/img/software/dynamixel/dynamixel_sdk/library_setup/java/linux/library_file/b1.webp)

# Building and Running the Sample Code
The DYNAMIXEL SDK example code for Java uses the library files(.so for Linux) built in C language.

Each released DYNAMIXEL SDK have latest library files in `[DynamixelSDK folder]/c/build/[linuxXX]/libdxl_xYY_c.so` which should be built by its own source code.

## Make Java Project

* Set workspace 

  ![](/img/software/dynamixel/dynamixel_sdk/library_setup/java/linux/sample_code/1.webp)

* Make new project. Click `New - Project...`

  ![](/img/software/dynamixel/dynamixel_sdk/library_setup/java/linux/sample_code/2.webp)

* Select `Java Project`, then click `Next`

  ![](/img/software/dynamixel/dynamixel_sdk/library_setup/java/linux/sample_code/3.webp)

* Type `Project Name`, then click `Next`

  ![](/img/software/dynamixel/dynamixel_sdk/library_setup/java/linux/sample_code/4.webp)

## Import function reference

* Click `Link Additional Source to Project`

  ![](/img/software/dynamixel/dynamixel_sdk/library_setup/java/linux/sample_code/5.webp)

* Click Browse

  ![](/img/software/dynamixel/dynamixel_sdk/library_setup/java/linux/sample_code/6.webp)

* Select the folder `linux64` in `c` language folder, then click `ok`  
  (for x86 platform, select `[DynamixelSDK folder]/java/dynamixel_functions_java/linux32`)

  ![](/img/software/dynamixel/dynamixel_sdk/library_setup/java/linux/sample_code/7.webp)

* Click `Finish`

  ![](/img/software/dynamixel/dynamixel_sdk/library_setup/java/linux/sample_code/8.webp)


## Import DynamixelSDK library

* Click `Link Additional Source to Project`

  ![](/img/software/dynamixel/dynamixel_sdk/library_setup/java/linux/sample_code/10.webp)

* Click Browse

  ![](/img/software/dynamixel/dynamixel_sdk/library_setup/java/linux/sample_code/11.webp)

* Select the folder `x64` in `java` language folder, then click `ok`

  ![](/img/software/dynamixel/dynamixel_sdk/library_setup/java/linux/sample_code/12.webp)

* Click `Finish`

  ![](/img/software/dynamixel/dynamixel_sdk/library_setup/java/linux/sample_code/13.webp)


## Import JNA

* Go to Libraries on the screen. 

  ![](/img/software/dynamixel/dynamixel_sdk/library_setup/java/linux/sample_code/14.webp)

* Click `Add External JARs`

  ![](/img/software/dynamixel/dynamixel_sdk/library_setup/java/linux/sample_code/15.webp)

* Choose the `jna-X.Y.Z.jar` downloaded previously.

  ![](/img/software/dynamixel/dynamixel_sdk/library_setup/java/linux/sample_code/16.webp)

* Click `Finish`

  ![](/img/software/dynamixel/dynamixel_sdk/library_setup/java/linux/sample_code/17.webp)


# Run example

* Make the port be able to be used

```bash
$ sudo chmod a+rw /dev/ttyUSB0
```

  ![](/img/software/dynamixel/dynamixel_sdk/library_setup/java/linux/sample_code/21.webp)

* Put `ReadWrite.java` source into the folder `src` on the screen

  ![](/img/software/dynamixel/dynamixel_sdk/library_setup/java/linux/sample_code/18.webp)

* Adjust the values surrounded with red circle to proper value as shown in the comment. 

  ![](/img/software/dynamixel/dynamixel_sdk/library_setup/java/linux/sample_code/19.webp)

* Click `Run` in the Eclipse toolbox above. Console window will show the results

  ![](/img/software/dynamixel/dynamixel_sdk/library_setup/java/linux/sample_code/20.webp)
