# Building and Running the Sample Code
DYNAMIXEL SDK example code for Python uses the library files(.so for Linux) built in C language.

Each released DYNAMIXEL SDK have latest library files in `[DynamixelSDK folder]/c/build/[linuxXX]/libdxl_xYY_c.so` which should be built by its own source code.

# Import libraries

* Click `Set Path`

  ![](/img/software/dynamixel/dynamixel_sdk/library_setup/matlab/linux/sample_code/1.webp)

* Click `Add with Subfolders...`

  ![](/img/software/dynamixel/dynamixel_sdk/library_setup/matlab/linux/sample_code/2.webp)

* Select `[DynamixelSDK folder]/c/include`

  ![](/img/software/dynamixel/dynamixel_sdk/library_setup/matlab/linux/sample_code/3.webp)

* Click `Add Folder...`

  ![](/img/software/dynamixel/dynamixel_sdk/library_setup/matlab/linux/sample_code/4.webp)

* Select `[DynamixelSDK folder]/c/build/linux64`  
  (for x86 platform, select `[DynamixelSDK folder]/c/build/linux32`)

  ![](/img/software/dynamixel/dynamixel_sdk/library_setup/matlab/linux/sample_code/5.webp)


# Import function reference

* Click `Add with Subfolders...`

  ![](/img/software/dynamixel/dynamixel_sdk/library_setup/matlab/linux/sample_code/6.webp)

* Select `[DynamixelSDK folder]/matlab`

  ![](/img/software/dynamixel/dynamixel_sdk/library_setup/matlab/linux/sample_code/7.webp)

* Click `Save` and then `Close`

  ![](/img/software/dynamixel/dynamixel_sdk/library_setup/matlab/linux/sample_code/8.webp)


# Run example

* Make the port be able to be used

```bash
$ sudo chmod a+rw /dev/ttyUSB0
```

  ![](/img/software/dynamixel/dynamixel_sdk/library_setup/matlab/linux/sample_code/16.webp)

* Click `Browse for folder`

  ![](/img/software/dynamixel/dynamixel_sdk/library_setup/matlab/linux/sample_code/9.webp)

* Select `[DynamixelSDK folder]/matlab`

  ![](/img/software/dynamixel/dynamixel_sdk/library_setup/matlab/linux/sample_code/10.webp)

* Double click `read_write.m`

  ![](/img/software/dynamixel/dynamixel_sdk/library_setup/matlab/linux/sample_code/11.webp)

* Adjust the values surrounded with red circle to proper value as shown in the comment. Then click `Run` in the Eclipse toolbox above. 

  ![](/img/software/dynamixel/dynamixel_sdk/library_setup/matlab/linux/sample_code/12.webp)

* Console window will show the results 

  ![](/img/software/dynamixel/dynamixel_sdk/library_setup/matlab/linux/sample_code/13.webp)
