# 제품 사양
- 무게 : 15g
- 크기 : 18mm x 36mm x 24mm
- 기어비 :  70 : 1
- 속력 : 150 RPM(at 3.7V)
- 특징
  - 양쪽 축 출력
  - 안전장치(클러치) 장착
