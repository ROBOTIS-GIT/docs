> 접촉 센서 TS-10

# 개요
- 올로(OLLO)에서 물체와의 접촉을 감지하는데 사용되는 감압식(스위치식) 센서입니다.
- 범퍼카처럼 물체와의 충돌을 감지할 필요가 있는 곳에 사용할 수 있습니다.

# 사양
- 무게 : 4g
- 크기 : 24mm x 18mm x 15mm

# 핀 배열 정보
![](/img/parts/sensors/ts-10_pinout.png)

1. 사용 안함
2. GND
3. ADC : 스위치 눌림 상태 출력
4. VCC
5. 사용 안함


:::warning

**주의**: 케이블을 잘못된 방향이나 무리한 힘으로 끼우거나 빼면 커넥터가 파손될 수 있으니 주의해주세요. 

:::
