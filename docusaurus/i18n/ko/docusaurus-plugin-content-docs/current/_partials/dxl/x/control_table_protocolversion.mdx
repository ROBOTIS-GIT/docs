장치의 프로토콜 타입 (DYNAMIXEL Protocol 1.0 또는 2.0)을 설정할 수 있습니다. 

함께 사용할 장치의 프로토콜 타입에 맞춰서 설정하세요.

|    값     |          설명          |                     호환되는 장치                     |
| :-------: | :--------------------: | :---------------------------------------------------: |
|     1     | DYNAMIXEL Protocol 1.0 | AX 시리즈, DX 시리즈, RX 시리즈, EX 시리즈, MX 시리즈 |
| 2(기본값) | DYNAMIXEL Protocol 2.0 |        MX-28/64/106(2.0), X 시리즈, PRO 시리즈        |




:::note

**참고** : 
- [다이나믹셀 프로토콜 2.0](/docs/dxl/protocol/protocol2)은 [프로토콜 1.0](/docs/dxl/protocol/protocol1)의 안전성을 대폭 개선하였습니다. 프로토콜 1.0을 사용할 경우, 컨트롤테이블의 일부 영역의 접근이 제한될 수 있습니다.
- 프로토콜을 변경하려면 [다이나믹셀 위자드 2.0](/docs/software/dynamixel/dynamixel_wizard2)을 사용하세요.
- 장치가 지원하는 프로토콜은 [프로토콜 호환표](/docs/popup/faq_protocol_compatibility_table)를 참조하세요.

:::
