Camera image calibration is not required in Gazebo Simulation.

<details>
<summary>
![](/img/icon_unfold.webp) **Click to expand : Camera Imaging Calibration with an actual TurtleBot3**
</summary>

1. Launch roscore on `Remote PC`.
```bash
$ roscore
```

2. Trigger the camera on `SBC`.
```bash
$ roslaunch turtlebot3_autorace_camera raspberry_pi_camera_publish.launch
```

3. Execute rqt_image_view on `Remote PC`.
```bash
$ rqt_image_view
```

      ![](/img/systems/turtlebot3/autonomous_driving/noetic_rpi_rqt_image_view.webp)

      > rqt_image view

</details>
